# PlantPulse — Global Project Rules
- **Tech Stack:** Flutter + Supabase + FastAPI (ML).
- **Domain:** Pakistani Agriculture (Tomato, Maize, Potato).
- **Core Strategy:** Causal Chain (Pest → Disease).

## Critical Guidelines
- **Tokens:** Be lean. Use `effortLevel: low`. Do not rewrite entire files for small changes.
- **Reference:** For detailed PRD, Features, and Architecture, always read `@project-docs/`.
- **Language:** English/Urdu bilingual support is a hard requirement.

## Commands
- **Test:** `flutter test`
- **Lint:** `flutter analyze`
